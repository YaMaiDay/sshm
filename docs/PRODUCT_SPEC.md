# 产品规划

## 产品目标

`sshm` 是一个全中文终端 TUI SSH 管理器，目标体验接近 OmnySSH，但避免内置远程终端导致的 Tab 补全冲突。

最终能力：

- 所有服务器管理操作都能在 TUI 中完成
- 实时查看全部服务器状态
- 快速登录服务器
- 上传/下载文件和目录
- 服务器配置统一保存在本工具配置文件中
- 不要求服务器安装额外软件

## 主界面

## 监控指标

每台服务器显示：

- 在线/离线
- CPU 使用率
- 内存使用率和用量
- Swap 使用率和用量
- 根分区磁盘使用率和用量
- Load average
- Uptime
- 系统版本
- Docker 运行容器数量
- systemd 失败服务数量
- 监听端口
- 最近错误

## 服务器管理

主界面支持：

- `a` 添加服务器
- `e` 编辑服务器
- `x` 删除服务器
- `m` 移动分类

添加/编辑服务器表单字段：

```text
分类
服务器名称
服务器地址
用户名
端口
密钥文件
跳板机
密码
```

保存到 `~/.config/sshm/servers.toml`：

```toml
[[servers]]
category = "production"
name = "demo-web"
host = "203.0.113.10"
user = "deploy"
port = 22
key_path = "/path/key.pem"
password = ""
```

## 登录

按 `Enter` 登录选中服务器。

登录时暂停 TUI，调用系统 `ssh`：

```sh
ssh Host
```

这样远程 shell 的 Tab 补全、vim、tmux、Ctrl+C 都保持原生行为。

## 文件传输

上传：

```text
选择服务器
按 u
选择本地常用目录
选择本地文件/目录
选择远程目录
自动上传
```

下载：

```text
选择服务器
按 d
选择远程目录
选择远程文件/目录
选择本地保存目录
自动下载
```

底层调用：

```sh
scp file host:/remote/
scp -r dir host:/remote/
scp host:/remote/file local/
scp -r host:/remote/dir local/
```

## 搜索、筛选、排序

支持：

- `/` 搜索服务器名称、IP、用户、分类
- `t` 按分类筛选
- `o` 只看在线
- `p` 只看异常
- `s` 切换排序字段
- `c` 清除筛选

排序字段：

- 分类 + Host
- 在线状态
- CPU
- 内存
- 磁盘
- Load
